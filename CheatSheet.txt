Cheat sheet

Create virtual environment:

IF Windows:
.\init.bat

IF Mac:
python3 -m venv venv
. venv/bin/activate

Running:

IF Windows:
.\run.bat

IF Mac:
. venv/bin/activate
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run

Front end Vue:
npm install -g @vue/cli
npm install
cd wabby_wabbo_crm
npm install

Compile:
cd wabby_wabbo_crm
npm run build
