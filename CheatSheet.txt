Cheat sheet

Create virtual environment:

IF Windows:
.\init.bat

IF Mac:
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt

Running:

IF Windows:
.\run.bat

IF Mac:
. venv/bin/activate
export FLASK_APP=flaskr
export FLASK_ENV=development
flask run

Front end Vue:
npm install -g @vue/cli
npm install
cd frontend
npm install

Compile:
cd frontend
npm run build

Testing frontend:
npm run serve
IN CASE OF SCRIPT IS NOT ENABLED ON THIS MACHINE:
Set-ExecutionPolicy RemoteSigned
